# TaskTracker - навчальний проєкт

## Мета проєкту
Навчитися робити базовий Angular застосунок "з нуля" на простому прикладі Task Tracker.

Фокус навчання:
- компоненти (standalone)
- data binding (зв'язок HTML <-> TypeScript)
- директиви для умов і циклів
- pipe для відображення даних
- локальне збереження у `localStorage`

## Що я хочу реалізувати
- Список задач: додати / видалити / позначити виконаною
- Фільтри: `All` / `Active` / `Done`
- Пошук задач
- Збереження задач у `localStorage`

## Ключові теми, які я відпрацьовую
- Standalone компоненти (`standalone: true`, `imports`)
- `@Input` / `@Output` і події між компонентами
- Цикли та умови в шаблоні:
  - новий синтаксис: `@for`, `@if`
  - старий синтаксис: `*ngFor`, `*ngIf`
- `ngClass` для умовних CSS-класів
- Pipe (наприклад, підсвітка тексту пошуку)
- Простий `TasksService` + робота з `localStorage`

## План розвитку по кроках
1. Додавання задачі (працює форма + рендер списку).
2. Позначення задачі виконаною (`done`).
3. Видалення задачі.
4. Фільтри `All/Active/Done`.
5. Пошук по назві задачі.
6. Збереження/читання задач з `localStorage`.
7. Виділення логіки в `TasksService`.
8. Додати pipe для підсвітки пошукового слова.

## Запуск
```bash
npm install
npm start
```

Відкрити в браузері:
- `http://localhost:4200/`
